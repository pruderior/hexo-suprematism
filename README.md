# Hexo Suprematism

A Hexo theme inspired by **Suprematism** and the **Russian Avant-garde** art movement, particularly the work of **Ilia Chashnik**.

## Features

- Bold geometric decorations (diagonal bars, circles, rectangles)
- Suprematist color palette: red, black, cream, yellow, blue
- Large typographic drop caps inspired by [Typology](https://demo.mekshq.com/typology/)
- Clean, text-focused layout
- Responsive design
- Smooth animations
- Code syntax highlighting with copy button

## Usage

Change your theme in `_config.yml`:

```yaml
theme: hexo-suprematism
```

## Theme Configuration

Edit `themes/hexo-suprematism/_config.yml` to customize:

- Navigation items
- Profile information  
- Post meta display options
- Date formats

## Inspiration

- Ilia Chashnik's "Study for Advertising Poster"
- Kazimir Malevich's Suprematist compositions
- Russian Constructivist typography
- Early Soviet avant-garde design
